# INDICATOR DEFINITIONS

Let P_t be closing price.

T40:
T_{40,t} = (1/40) * sum_{i=0}^{39} P_{t-i}

Delta T40:
ΔT_{40,t} = T_{40,t} - T_{40,t-1}

EMA 26:
T_{26,t} = α_26 P_t + (1 - α_26) T_{26,t-1}
α_26 = 2/27

EMA 12:
T_{12,t} = α_12 P_t + (1 - α_12) T_{12,t-1}
α_12 = 2/13

D_t:
D_t = T_{12,t} - T_{26,t}

RSI(14), Wilder smoothing:
α = 1/14

Define:
ΔP_t = P_t - P_{t-1}
G_t = max(ΔP_t, 0)
L_t = max(-ΔP_t, 0)

AvgGain_t = α G_t + (1-α) AvgGain_{t-1}
AvgLoss_t = α L_t + (1-α) AvgLoss_{t-1}

RS_t = AvgGain_t / AvgLoss_t

RSI_t = 100 - 100 / (1 + RS_t)

Eligibility:
E_t = 1 if (P_t > T_{40,t} AND ΔT_{40,t} > 0)

Tactical states:
T1: D_t >= 0 AND RSI_t >= 50
T2: D_t >= 0 AND RSI_t < 50
T3: D_t < 0 AND RSI_t >= 50
T4: D_t < 0 AND RSI_t < 50

Concordance:
C_t = 1 if aligned (T1 or T4)

Run length:
k_C(t) increments while C_t=1, else 0

Transition density:
Count of transitions in last m=10 bars:
- Eligibility toggle
- D_t crossing 0
- RSI crossing 50
